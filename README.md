# Insurance Analytics Dashboard

A modern, interactive dashboard for analyzing insurance data with predictive modeling capabilities.

## Features

- Interactive data visualization
- Predictive analytics
- Responsive design (desktop & mobile)
- Real-time data updates

## Setup

1. Clone this repository
2. Open `index.html` in a web browser
3. Data is loaded from `data/analytics.json`

## Structure

```
insurance-analytics/
├── css/
│   └── style.css          # Dashboard styles
├── js/
│   └── dashboard.js       # Main dashboard logic & predictive models
├── data/
│   └── analytics.json     # Data source
└── index.html             # Main dashboard page
```
